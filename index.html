My Projects

Here is some of the projects I have done

  • Disk Allocation Method(Java)

    This project uses MVC architecture in simulating different disk allocation methods: Contiguous, Chained and Indexed by storing a file from the real system to the simulated disk, displaying file table, and displaying the file by reading blocks from the disk.

  • Reversi Game(Java)

    This project resembles a simple version of Reversi game that have options for user to play with computer or with another player. The game goal is to score as much as possible by taking turn putting disks to the appropriate positions on the board. The game end if there are no more moves to be made.

  • Chat Room(C, SQLite)

    This is a project for UNIX class, running on UNIX server. The project is a resemble of a chat room that allows Client to connect to Server to send and receive messages. The Client can also send request for data in a database and the Server will query the database and send back the information. At most 8 clients can connect and extract data from the server at the same time.

  • Book Finder(C#, SQL Server)

    A WinForms application that allows users to find the store carrying a book and the number of available copies by looking into a database simulating a bookstore series. Users can search by one or more information about the book and choose to order the list of stores by location or number of copies.

  • Secret Diary (HTML, CSS, Bootstrap, JavaScript, jQuery, PHP, MySQL)

    This project is a website that allow users to record their every day’s thoughts by logging them into their accounts and automatically record their input into a database using MySQL. Users can keep logging in by the use of cache.

  • Website Skeleton (HTML5, CSS, JavaScript, Bootstrap, Thymeleaf, jQuery, Stripe, Spring Boot, AWS)

    A website skeleton that is ready to sell any ideas online. With the two ways of subscription: Basic and Pro, the website allows user to log in/ sign up, check the company information as well as the selling content, and also contact the support to send feedback.
